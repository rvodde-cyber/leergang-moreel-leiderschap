# Eerlijk Oordeel

**Waar sta jij als ethisch leider?**

Zelfreflectie-instrument — geen quiz, geen cijfer. Vijf dimensies (Hoofd → Ruggegraat), Chart.js-radar, print/PDF.

## Starten

Via hoofdapp: **Moreel Leiderschap → Eerlijk Oordeel**

Of lokaal:

```bash
cd instruments/eerlijk-oordeel
npx serve .
```

Open `http://localhost:3000`
